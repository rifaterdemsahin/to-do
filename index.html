Self-Learning Projects Boilerplate

A structured 7-stage framework bridging the gap between problem discovery, environment preparation, logic formulation, and validated deployment.

The 7-Stage Framework

  • 1

    Real Unknown

    Identifying core problem statements, OKRs, and assumptions to validate.

  • 2

    Environment Context

    Preparing macOS, Windows setups, credentials managers and local AI tools.

  • 3

    Simulation

    Formulating user journeys and mockups before writing production code.

  • 4

    Formula

    Creating decision logs, implementation recipes, and LLM thinking logs.

  • 5

    Symbols

    Compiling main code, Docker configurations, and Git workflows.

  • 6

    Semblance

    Chronological logs of errors, hotfixes, and objective-to-reality gaps.

  • 7

    Testing Known

    Close-loop validation maps, sign-offs, and master checklist audits.
